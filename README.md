Odontologia Reishtatter — Sistema de Gestão

MVP desenvolvido como projeto do 3º Semestre do curso de Tecnologia em Informática para Negócios.

DESCRIÇÃO
Sistema para gestão de uma clínica odontológica, contemplando o cadastro de pacientes,
profissionais e secretárias, agendamento e registro de consultas, controle de
tratamentos com etapas, geração de orçamentos e controle de status.

TECNOLOGIAS UTILIZADAS
- Java (Linguagem de Programação I)
- SQL Server (Banco de Dados)
- HTML / CSS / JavaScript (Interface Gráfica)

Como Executar:

BANCO DE DADOS
1. Abra o SQL Server Management Studio (SSMS)
2. Execute o arquivo `database/clinica_bdsql.sql`
3. O banco `clinica_db` será criado com todas as tabelas e dados de exemplo

JAVA
1. Importe o projeto em sua IDE (Eclipse, IntelliJ ou VS Code)
2. Execute a classe `App.java`

INTERFACE
1. Abra o arquivo `interface/clinica_mvp.html` em qualquer navegador

FUNCIONALIDADES
- Cadastro de Pacientes, Profissionais e Secretárias
- Agendamento e registro de Consultas
- Controle de Tratamentos e Etapas
- Geração e aprovação de Orçamentos
- Controle de status em todas as entidades
